Web Components

Construyendo nuestro HTML5

Descripción

Web Components permiten a los desarrolladores crear sus propias etiquetas HTML personalizadas, dejarles usar esas etiquetas en sus sitios y aplicaciones, y permiten la reutilización de elementos más fácil.

Ejemplos

¿Como creo mi propio Web Element?

var miElemento = document.registerElement('mi-elemento');
<mi-elemento></mi-elemento>

Naming rules

You need to have at least one '-' inside the name of a custom element. Any tag names without '-' will result in an error.
